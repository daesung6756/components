Class guide

* 추가 클래스 아무것도 넣지 않으면 일반 레이어 팝업 사용.

  • is-dimmed : 딤 레이어 팝업 사용
  • is-full : 모바일에서 전체 화면 팝업
  • is-confirm : 컨펌 디자인 팝업
  • is-alert : 경고창 디자인 팝업

Data name Guide

  • data-popup : 팝업 마크업
  • data-popup-open : 팝업 열기.
  • data-popup-close : 팝업 닫기.

데이터명 규칙은 자율이지만 팝업 타겟 / 팝업 오픈 / 팝업 클로즈 데이터는 동일한 네이밍을 해야 합니다. 한 세트 입니다. 추가적인 이벤트를 처리하기 위해서는 onclick 활용 해주세요.
